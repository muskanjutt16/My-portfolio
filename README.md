## 🚀 How to Run Locally

Want to see this project on your computer? Follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/muskanjutt16/My-portfolio.git
```

1. Navigate to the folder
   ```bash
   cd My-portfolio
   ```
2. Open the HTML file
   · Simply double-click index.html OR
   · Use Live Server in VS Code

That's it! No dependencies or installations required. 🎉
